# Lesson01：股票是什么？为什么要量化投资？

## 本课目标

完成本课后，你应该能够：

- 用自己的话解释股票代表什么；
- 区分企业价值、股票价格和投资收益；
- 解释量化投资解决什么问题，以及它不能解决什么问题；
- 建立“假设—数据—规则—回测—风控—复盘”的基本研究流程。

## 1. 股票是什么？

股票是股份公司所有权的一部分。购买股票，意味着你取得了企业极小的一份权益，并相应承担企业经营和市场价格波动带来的风险。

股票价格是市场参与者在某一时刻达成的交易价格，不等同于企业的内在价值。短期价格会受预期、流动性、情绪和事件影响；长期回报则更依赖企业盈利、现金流、资本配置与买入估值。

## 2. 投资收益来自哪里？

股票投资的长期收益通常来自：

1. 企业盈利增长；
2. 分红与股份回购；
3. 市场给予的估值变化。

估值提升可以带来收益，也可能反向压缩收益。因此，“好公司”并不自动等于“好价格”。

## 3. 为什么要量化投资？

量化投资是把投资假设转化为明确规则，使用历史数据检验，并以一致的方式执行和复盘。它的价值主要在于：

- 同时处理大量标的和数据；
- 减少临场情绪对决策的干扰；
- 在投入资金前暴露策略缺陷；
- 让研究过程可重复、可比较、可迭代。

量化模型不能保证盈利。数据偏差、过拟合、交易成本、市场制度和策略失效，都可能让回测与实盘表现不同。

## 4. 第一套研究框架

```text
提出假设
  ↓
准备数据
  ↓
定义信号与交易规则
  ↓
加入成本和市场约束
  ↓
回测与样本外验证
  ↓
评估收益、回撤和稳定性
  ↓
小规模模拟与复盘
```

## 5. 本课纪律

- 不因为亏损而自动认定“会涨回来”；
- 不因单次盈利认定策略有效；
- 不只看年化收益，同时检查最大回撤、波动和交易成本；
- 在交易前写清买入、卖出和策略失效条件；
- 初学阶段以研究和模拟为主，不急于自动实盘。
