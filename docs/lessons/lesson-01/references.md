# Lesson01 参考资料

本课先以概念理解和独立表达为主，不要求大量阅读。后续课程将逐步补充交易规则、财务报表、指数与回测资料。

## 建议阅读方向

- 中国证监会投资者教育材料：证券市场基础知识与投资风险；
- 上海证券交易所、深圳证券交易所：股票与 ETF 交易规则；
- 上市公司定期报告：理解股东权益、利润与现金流；
- 指数公司编制方案：理解指数样本选择与调整；
- 回测方法资料：前视偏差、幸存者偏差、过拟合与交易成本。
