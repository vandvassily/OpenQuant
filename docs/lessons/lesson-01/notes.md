# Lesson01：股票是什么？为什么要量化投资？

## 学习目标

建立正确的投资认知，理解股票本质，并确定 OpenQuant 的投资方向。

## 核心认知

股票不是一个价格代码，而是一家公司所有权的凭证。

长期股票收益的核心来源：

- 企业价值增长
- 盈利能力提升
- 未来现金流增长

股票价格反映市场对企业未来价值的预期。

## 投资方向

采用混合模式：

价值投资 + 量化投资 + AI辅助研究

## 我的投资理念

1. 不买看不懂的企业。

2. 不买没有盈利能力的企业。

3. 量化交易的投资决策，必须经过数据验证。

## Lesson总结

投资不是预测短期涨跌，而是建立可验证、可持续迭代的投资系统。
